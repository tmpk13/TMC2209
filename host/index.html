Canvas / drag

Drag joints in the canvas; shift snaps to 15°. 0 homes, space stops. Middle-drag pans, scroll zooms. The canvas visualises the first three linkages only (J0..J2); J3 (base rotation) and J4 (linear stage) live in the Joints tab.

Per-joint jog & targets

Click a jog button to nudge by the selected size. The "go" button sends the absolute target value in the input next to it. The "set" button under each joint re-labels that microstep value as the current position without commanding any motion.

End-effector servos

Pulse widths are in microseconds. Deadzone ignores moves smaller than the listed us; speed limits how fast the commanded pulse slews to the target. The "home on enable" flag tells the firmware to snap the target to home us on the next config push - useful for safe startup.

TMC2209 config (UART)

Toggling a switch sends SetTmcConfig (tag 0x05); the badge reads GCONF+CHOPCONF back over UART (tag 0x07) so you can confirm the chip actually latched the change. J4 sits in standalone mode (no UART driver) and is therefore not shown here.

URDF inverse kinematics

Loads a URDF + motor map (defaults to host/urdf/robot.urdf and host/urdf/motor_map.json) and solves IK with ikpy. Coordinates are in metres, robot base frame.

USB joystick

      

        Reads /dev/input/jsN. Analog axes command motor
        velocity directly: stick deflection x sensitivity sets vmax for
        joints (steps/s) and gates motion for servos (which travel at their
        configured speed). Releasing the stick stops the motor at its current
        position - the target does not accumulate. Buttons still add their
        sensitivity (in target units) once per press.
